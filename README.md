# Manae Shopping Mart LLC

**Live Site:** [https://manaeshoppingmartllc.com](https://manaeshoppingmartllc.com)

Manae Shopping Mart LLC is a fully functional e-commerce website built to provide a seamless online shopping experience. The platform supports product browsing, user accounts, shopping cart, and checkout functionalities.

## Features

- User registration and authentication
- Product catalog with categories
- Add to cart and checkout system
- Payment integration
- Order management for users and admin
- Responsive design for desktop and mobile
- Search and filter products
